# AdamPicoPSU
Yet another Coleco Adam PicoPSU, cheap and compact

It use a 3 amp 12v regulated PSU for input, and a PICOPSU 20 pin for generate the additional -12v and -5v. A lm7905 converts the -12v to the needed -5v too.
